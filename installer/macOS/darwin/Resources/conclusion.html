AIR-Tools

Thank you for installing AIR-Tools.

Run AIR-Tools

Open a new terminal and run the following command to get start with AIR-Tools:

  $ air-tools [command]

Uninstall AIR-Tools

Run the following command to uninstall AIR-Tools.

  $ sudo bash /Library/__PRODUCT__/__VERSION__/uninstall.sh
